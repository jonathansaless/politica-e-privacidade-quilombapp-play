Política de Privacidade

Como o QuilombApp Play cuida dos seus dados

Esta página explica de forma clara como coletamos, utilizamos, armazenamos e protegemos as informações dos usuários do jogo QuilombApp Play.

Ao utilizar o jogo, você concorda com os termos desta Política de Privacidade. Caso não concorde, recomendamos que interrompa o uso do aplicativo.

Plataforma: Android Base legal: LGPD (Lei nº 13.709/2018) Última atualização: 04/12/2025
1. Quem somos

1. Quem somos

O QuilombApp Play é um jogo educacional desenvolvido com o objetivo de divulgar e valorizar a cultura quilombola por meio de uma experiência interativa.

Esta Política de Privacidade é mantida pelo responsável pelo desenvolvimento do jogo (doravante chamado de “Desenvolvedor”).

2. Dados que coletamos

2. Quais dados coletamos

Ao utilizar o QuilombApp Play, podemos coletar os seguintes dados:

2.1. Dados de conta

  • E-mail (em contas com registro por e-mail e senha);
  • Apelido (nickname) utilizado no jogo;
  • Idade (informada no cadastro);
  • ID de usuário gerado automaticamente pelo Firebase.

2.2. Dados de jogo

  • Quantidade de vidas e moedas disponíveis;
  • Itens e recursos obtidos (por exemplo: dicas, poderes, avatares desbloqueados);
  • Progresso no jogo (pontuações, desafios concluídos e estatísticas de uso dentro do app).

2.3. Dados técnicos

De forma automática, a infraestrutura que utilizamos (como o Firebase) pode coletar dados técnicos, por exemplo:

  • Modelo do dispositivo e sistema operacional;
  • Identificadores técnicos gerados pelo Firebase;
  • Informações de erro (logs) para fins de depuração e correção de bugs.

Não coletamos deliberadamente dados sensíveis (como origem racial ou étnica, religião, opinião política, saúde etc.) por meio do QuilombApp Play.

3. Como usamos seus dados

3. Como utilizamos os dados

Os dados coletados são utilizados para:

  • Criar e gerenciar sua conta no jogo;
  • Sincronizar o progresso do jogo entre dispositivos (quando conectado à internet);
  • Exibir corretamente vidas, moedas, avatares e demais informações no menu;
  • Manter rankings, estatísticas internas e recursos de personalização do jogador;
  • Melhorar a estabilidade, desempenho e segurança do aplicativo.

Não utilizamos seus dados pessoais para fins de marketing, envio de spam ou venda de informações para terceiros.

4. Login, visitante e internet

4. Login, modo visitante e conexão à internet

4.1. Login com e-mail e senha

Ao optar por criar uma conta com e-mail e senha, o jogo utiliza Firebase Authentication para autenticar sua identidade de forma segura.

4.2. Login como visitante

O jogo também oferece a opção de login como visitante, sem necessidade de e-mail. Nesse caso, um identificador anônimo é criado para salvar seu progresso.

4.3. Falta de conexão

Alguns recursos do jogo dependem de conexão com a internet (por exemplo, login, sincronização de progresso e carregamento de dados do jogador). Quando não há conexão, o jogo exibe mensagens informando a impossibilidade temporária de acessar esses recursos.

5. Bases legais e LGPD

5. Base legal para tratamento de dados (LGPD)

O tratamento dos seus dados pessoais é realizado em conformidade com a Lei Geral de Proteção de Dados Pessoais (LGPD – Lei nº 13.709/2018), principalmente com base em:

  • Execução de contrato: para permitir o funcionamento do jogo e dos seus recursos;
  • Consentimento: quando você aceita esta Política e utiliza o aplicativo;
  • Legítimo interesse: para melhorar a experiência, corrigir erros e garantir segurança.
6. Compartilhamento de dados

6. Compartilhamento de dados com terceiros

O QuilombApp Play não vende nem compartilha seus dados pessoais com terceiros para fins comerciais.

Contudo, utilizamos serviços de terceiros para hospedar e processar dados, como:

  • Firebase Authentication – autenticação de usuários;
  • Firebase Cloud Firestore – armazenamento seguro de dados de jogo.

Esses serviços são fornecidos pela Google LLC e possuem suas próprias políticas de privacidade, que recomendamos consultar.

7. Segurança

7. Segurança dos dados

Adotamos boas práticas de segurança compatíveis com as ferramentas utilizadas (como o Firebase) para proteger seus dados contra acessos não autorizados, alteração ou destruição indevida.

Ainda assim, nenhum sistema é 100% seguro, e não é possível garantir proteção absoluta contra todos os tipos de ataque. Em caso de incidente grave envolvendo dados pessoais, buscaremos notificá-lo pelos meios disponíveis.

8. Crianças e adolescentes

8. Crianças e adolescentes

O QuilombApp Play pode ser utilizado por pessoas de diferentes faixas etárias. Recomendamos que crianças e adolescentes utilizem o jogo com acompanhamento de responsáveis, especialmente no momento de criação de contas e fornecimento de dados.

Caso você seja pai, mãe ou responsável e acredite que possamos ter coletado dados pessoais de uma criança sem o devido consentimento, entre em contato conosco para solicitar revisão ou exclusão.

9. Seus direitos

9. Seus direitos como titular de dados

Nos termos da LGPD, você possui, entre outros, os seguintes direitos:

  • Confirmar se tratamos dados pessoais a seu respeito;
  • Acessar os dados pessoais que mantemos sobre você;
  • Corrigir dados incompletos, inexatos ou desatualizados;
  • Solicitar anonimização, bloqueio ou eliminação de dados desnecessários;
  • Solicitar a exclusão da sua conta e dos dados associados ao jogo;
  • Revogar o consentimento, quando esta for a base legal utilizada.

Para exercer qualquer um desses direitos, entre em contato pelos canais descritos abaixo.
